# Карточки по лекции 1

Статический mobile-first тренажер по первой лекции: NumPy, Pandas и базовый анализ данных.

## Формат

- Билет: 20 случайных вопросов.
- Накопительный прогресс: правильный ответ добавляет вопрос к освоенным.
- Повторная ошибка по ранее освоенному вопросу снимает его из прогресса.
- Полный тест открывается после 130 освоенных вопросов.

## Как открыть локально

Откройте `index.html` в браузере.

## Как развернуть на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое папки `lecture01-cards` в корень репозитория.
3. В настройках репозитория откройте `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. После публикации GitHub Pages даст ссылку на сайт.

Проект не требует сборки, Node.js или внешних библиотек.
